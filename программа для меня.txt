Описание интерфейса:

Что нам нужно?

Для начала - скеттерплот по двум признакам с цветовой
дифференциацией точек по подкорпусам

1. Задаём подкорпус по экстралинг. признакам:
sex (m, f, не выбран) - поле sex, input селект
date (от) - поле date_start, input Date
date (до) - поле date_end, input Date
mark (от) - поле mark_start, input text
mark (до) - поле mark_end, input text

Рещение (шаг 1):
Сначала бэкенд находит все объекты типа Document соответствующие этим параметрам
Формируем разные наборы объектов типа Document для каждого набора признаков

2. Выбираем два признака
Признак 1:
токен - поле var1-token__text, input text
лемма - поле var1-token__lemma__text, input text
часть речи - поле var1-token_lemma__pos, input селект
Признак 2:
токен - поле var2-token__text, input text
лемма - поле var2-token__lemma__text, input text
часть речи - поле var2-token_lemma__pos, input селект

Решение (шаг 2):
Для каждого документа в каждом наборе подсчитываем число сущностей типа 1 и сущностей типа 2
Откладываем каждый набор документов точками на графике отдельным цветом
Считаем корреляцию между признаками и уравнение регрессии для каждого отдельного набора и для объединённого набора
Всё это объединяем в одну Html-страницу, отправляем её пользователю и открываем её в отдельном окне.
